HTML

  • HTML stands for Hypertext Markup Language
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • CSS stands for Cascading Style Sheets
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • Git Flow:
    • git checkout -b *insert branch name here* : checks what branch we are currently on
    • git add -A : stages all current changes to be committed
    • git commit -m "*insert commit message here*" : commits all changes to be pushed
    • git pull origin main : pulls from main branch to make sure we are working on the most up to date version of the project
    • git push origin *insert branch name here* : pushes commit to our main branch to be merged into the repository
